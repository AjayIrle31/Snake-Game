# Snake-Game
Classic snake game made with just HTML,CSS and JAVASCRIPT.

## Features

- Smooth snake movement with arrow key controls
- Score tracking — earn a point for every food eaten
- Timer — tracks how long you survive
- High score — saved in localStorage, persists across sessions
- Game Over screen with restart option
- Food never spawns on the snake's body

## How to Play

1. Open `index.html` in your browser
2. Click **Start Game**
3. Use arrow keys to move the snake
4. Eat the green food to grow and score points
5. Avoid hitting the walls or yourself

## Controls

| Key | Action |
|-----|--------|
| ← Arrow | Move Left |
| → Arrow | Move Right |
| ↑ Arrow | Move Up |
| ↓ Arrow | Move Down |

## Tech Stack

- HTML

- JavaScript (Vanilla)
